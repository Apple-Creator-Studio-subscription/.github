# Apple Creator Studio — Portable Professional Creative Suite Subscription

## About Apple Creator Studio

Apple Creator Studio provides a complete professional creative environment for users who need Apple's pro apps in one subscription. Apple Creator Studio combines Final Cut Pro (video editing), Logic Pro (music production), Motion (motion graphics), Compressor (video encoding), and MainStage (live performance). Apple Creator Studio enables users to access all apps for one monthly price. Apple Creator Studio runs on desktop platforms. Apple Creator Studio is trusted by video editors, music producers, motion designers, and creative professionals worldwide.
